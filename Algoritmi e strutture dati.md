## Strumenti necessari:

Serve a svincolarsi dalla specifica implementazione di un linguaggio di programmazione specifico.

## Problem:
A problem is a task to be performed.
- It should be precisely defined and thoroughly understood.
- It should include constraints on the acceptable quantity of resources that may be consumed by the solution.

## Algorithm:
An algorithm is the process followed to solve a problem. 
there can be many algorithms for solving one problem.

It needs to have some properties:
1. It must be correct (solve the problem correctly)
2. It's composed of a series of concrete and defined steps, doable in a finite amount of time
3. There can be no abiguity on which step will be performed next
4. It must be composed of a finite number of steps
5. It must terminate

## Program:
A computer program is an instance, or concrete rappresentation, of an algorithm in a programming language.
